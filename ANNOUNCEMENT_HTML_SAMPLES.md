# Announcement HTML Content Samples

Here are some sample HTML snippets you can use for announcements. These can be used in the `description` field or if you need rich formatting, you can add HTML content support back later.

    We're excited to announce our <strong style="color: #f97316;">Summer Special</strong>!

  </p>
  <ul style="list-style-type: disc; margin-left: 25px; margin-bottom: 20px; line-height: 2;">
    <li>Get 20% off on all steaks</li>
    <li>Buy 2, Get 1 free on breakfast meals</li>

## Usage Notes:

1. **For Description Field**: You can use plain text or simple HTML. The description field will render HTML if you include it.

2. **Styling Tips**:
   - Use inline styles (as shown above) for better compatibility
   - Keep colors consistent with your brand (green #22c55e, orange #f97316)
   - Use emojis sparingly for visual appeal
   - Keep text readable with good line-height (1.6-1.8)

3. **Best Practices**:
   - Keep announcements concise and clear
   - Use bold text for important information
   - Include clear call-to-action if applicable
   - Always include validity dates for promotions

4. **If you want to add HTML content back**: You can add a separate `content` field that supports rich HTML formatting, while keeping `description` for simple text summaries.
